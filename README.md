## API Endpoints

- **POST** `/tickets`: Create a new ticket.
- **GET** `/tickets`: Retrieve all tickets.
- **GET** `/tickets/:id`: Retrieve a single ticket by ID.
- **PUT** `/tickets/:id`: Update a ticket by ID.
- **DELETE** `/tickets/:id`: Delete a ticket by ID.
